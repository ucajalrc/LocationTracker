
# Location Heat Map App

## Overview

The Location Heat Map App is a cross-platform mobile application built using **C# and .NET MAUI**.  

The application tracks the user's real-time GPS location, stores the location data in a local **SQLite database**, and visualizes the stored locations on a map as a heat map.

The heat map is generated by drawing semi-transparent circles on the map for each saved location. When locations overlap, the color intensity increases, simulating a heat map effect.

---

## Technologies Used

- C#
- .NET MAUI
- .NET MAUI Maps
- SQLite (sqlite-net-pcl)
- Android Emulator
- Visual Studio Code

---

## Features

- Requests user location permission
- Tracks device GPS location
- Saves location data locally using SQLite
- Displays saved locations as heat map circles
- Automatically updates the map in real time
- Centers map on most recent location

---

## How the Heat Map Works

Instead of using traditional map pins, the application:

1. Stores latitude and longitude values in SQLite.
2. Retrieves all saved location records.
3. Draws a semi-transparent `Circle` on the map for each location.
4. Overlapping circles create stronger visual intensity.

This produces a heat map visualization effect.
